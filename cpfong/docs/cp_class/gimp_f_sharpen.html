2. 銳利（Sharpen）



濾鏡名稱路徑功能說明

Sharpen (Unsharp Mask)Filters → Enhance → Sharpen (Unsharp Mask)增加邊緣對比，提升清晰度

High PassFilters → Enhance → High Pass保留細節，常搭配圖層混合模式使用

Wavelet Sharpen插件或進階套件 (如 G’MIC)高品質照片銳化



HSV: 



H  色相 Hue  顏色種類（如紅、綠、藍）以 0~360 度表示

S  飽和度 Saturation 顏色的純度，0 為灰色，100 為最鮮明

V  明度 Value 明亮程度，0 為黑，100 為最亮的顏色


範例應用：
     加強焦點區
    曝光不足造成模糊時補救
    網頁圖示或字體銳化處理

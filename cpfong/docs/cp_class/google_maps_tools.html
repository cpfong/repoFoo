🔄 1. 復原 / 重做
左箭頭：復原上一步操作。
右箭頭：重做剛剛復原的操作。


✋ 2. 手形工具（移動畫面）
點選後，用滑鼠 按住地圖拖曳，即可移動畫面。
當你只想查看或移動地圖而不進行編輯時使用。


📍 3. 新增地標（標記）
點選圖示後，在地圖上點一下想標記的位置。
可輸入名稱與描述，標記會儲存在目前的圖層中。


🔗 4. 繪製線條或形狀
點下去後，選擇：
「新增線條或形狀」可畫出路線或區域。
在地圖上點選多個點來構成路線，雙擊最後一點以完成繪製。


➰ 5. 加入路線（開車、騎車、步行）
點選後，可選擇交通方式。
然後在地圖上選擇起點與終點，系統會自動規劃路線。


📏 6. 測量距離與面積
點選後，在地圖上依序點選兩個或多個點。
系統會顯示每段距離與總長度；若封閉則會顯示面積。

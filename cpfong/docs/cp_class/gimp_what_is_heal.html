它和 Clone Tool 類似，但更聰明、更自然融合。

🩺 Heal Tool 是什麼？

Heal Tool（修復工具，快捷鍵 H） 是一種智慧型修補筆刷。它的核心功能是：

✅ 從來源區複製像素，但融合目標區的紋理、光影、顏色，產生自然過渡。
簡單說，它會「修得像沒發生過一樣」。

🔍 Heal Tool vs Clone Tool

功能	Clone Tool（複製）	Heal Tool（修復）
複製來源	✅ 直接複製來源像素	✅ 複製來源像素
是否融合目標區	❌ 否，像素硬貼	✅ 會根據目標區自動融合
結果外觀	📦 明顯重複、易看出「貼過」	🌿 看不出痕跡，像本來就長那樣
適合場景	結構明確（建築、物件、髮絲）	自然紋理（皮膚、天空、背景過渡）

✅ 方法一：使用「Text: Simple」濾鏡（推薦）
在時間軸放好你要加入文字的片段
通常放在 Video 軌（V1, V2 等）
選取該片段 → 點選上方 Filters → + 按鈕
搜尋並加入「Text: Simple」
顯示文字輸入框
可自訂：文字內容、字型、大小、顏色、對齊方式、背景透明度等
位置調整
同樣在 Filters 加入 Size, Position & Rotate 濾鏡，精準控制文字出現位置
加上動畫（選用）
套用 Fade In Video、Fade Out Video 等濾鏡，讓文字平滑出現／消失

✅ 方法二：獨立文字片段（標題畫面）
如果你想要一個「只有文字、無影片」的標題畫面：

上方選單 Open Other → Text
輸入文字內容 → 點 OK
新增一筆文字片段到 Playlist，可重複使用
拖曳到時間軸（建議放在 V2），並調整長度

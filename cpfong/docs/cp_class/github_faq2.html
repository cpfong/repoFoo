Q: 可以在教室使用主機上網申請帳號否?
A: 不妥
✅ 1️⃣ 開啟註冊頁面 https://github.com/signup

👉 前往：
✅ 2️⃣ 填寫註冊資料

    Username（使用者名稱）：

        建議用英文、數字組合，例如 johnchen2025

        ⚠️ 不可重複、不能有空白或特殊符號。

    Email address（電子郵件）：

        請填寫有效且可收信的 email（Gmail、Yahoo、Outlook 等皆可）。

    Password（密碼）：

        至少 8 個字元，包含大小寫字母、數字、符號。

        例如：Abc12345!

✅ 3️⃣ 驗證人機身份

    系統會出現簡單的 驗證挑戰（Captcha），照指示完成。

✅ 4️⃣ 選擇偏好

    Do you want to receive product updates and announcements via email?

        自由選擇（可打勾或不打勾）。

點【Create account】送出。
✅ 5️⃣ Email 驗證

    前往你填寫的 email 信箱，查找來自 GitHub 的驗證信。

    點擊郵件內的【Verify email address】按鈕完成驗證。

⚠️ 注意：

    驗證信件若未收到，請檢查垃圾郵件匣。

    驗證必須完成，否則帳號無法正常使用。

✅ 6️⃣ 略過不必要設定

註冊後會引導你選擇一些偏好（例如使用目的、功能介紹等），你可以直接略過或依個人喜好設定，最後進入主畫面即可。
✅ 7️⃣ 確認帳號正常

    登入 GitHub 主頁，右上角應該看到你的帳號名稱。

    點選個人頭像 → 【Settings】檢查 email 已顯示為 Verified 狀態。

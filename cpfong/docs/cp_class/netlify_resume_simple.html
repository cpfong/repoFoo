🌐 目標：

教學生用 Netlify + GitHub 模板快速建立個人首頁。

🚀 步驟教學範例：

1️⃣ 準備工作：
學生需要：
✅ 一個 GitHub 帳號（已具備）
✅ 一個 Netlify 帳號：

前往 https://www.netlify.com/
點選 Sign up
選擇用 GitHub 帳號登入（最快最簡單）


2️⃣ 選擇簡單的模板：
我們用一個簡單、純 HTML/CSS 的模板來練習。
🔗 範例模板：https://github.com/startbootstrap/startbootstrap-resume
這個模板是個人履歷頁面，簡單易懂、易修改。


3️⃣ Fork 模板：
讓學生打開這個模板的 GitHub 頁面。
點擊右上角的 Fork，建立自己的副本（存在自己帳號底下）。


4️⃣ 連結 Netlify 與 GitHub：
登入 Netlify。
點選 「Add new site」 → 「Import an existing project」。
選擇 GitHub 作為來源。
授權 Netlify 存取 GitHub（如果第一次操作會跳出授權畫面）。
選取剛剛 Fork 出來的個人倉庫。


5️⃣ 設定建置：
Build command（建置指令）：（留空，因為是純 HTML）
Publish directory（發布目錄）： .（根目錄）
點選 Deploy Site（部署網站）。


6️⃣ 完成 - 網站上線：
Netlify 會自動給一個臨時網址，例如：
https://brave-torvalds-12345.netlify.app
學生馬上可以瀏覽自己做好的網站。


7️⃣ 客製化個人首頁：
回到 GitHub 倉庫：
編輯 index.html
改上自己的姓名
放上自己的照片
加入簡短的自我介紹
提交並推送（Commit & Push）
Netlify 會自動更新網站內容。
✅ 進階提示（可選）：
修改網址名稱：
到 Netlify → Site settings → Change site name
改成像是：
https://john-doe-homepage.netlify.app
綁定自訂網域（有域名的話）：
Netlify → Domain Settings 裡設定自訂網址。


🔑 給學生的重要觀念：
Netlify 會自動監控 GitHub 倉庫，只要推送更新，網站就會自動改版。
這是典型的靜態網站，沒有後端，適合初學者。
編輯 index.html 能快速學會 HTML 結構與標籤。

點選右側面板下方「高級」可自訂以下選項：

1. 格式（Format）
常見選擇：
mp4：最常用，支援廣泛
mov：蘋果系統友好
mkv：支援多音軌、字幕

2. 編碼器（Codec）
libx264：H.264（通用）
libx265：H.265（較高壓縮）
mpeg4：老設備用

3. 解析度與幀率
會依據「專案設定」自動設定，若要修改：
在「視頻」標籤內調整「解析度（Resolution）」與「幀率（Frame rate）」
建議 YouTube 為：1920x1080 @ 30fps

4. 音訊（Audio）
可設定：
編碼器（aac, mp3）
比特率（一般 128k 或 192k）
取樣率（一般用 48000 Hz）

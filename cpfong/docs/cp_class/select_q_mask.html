🎭 什麼是「快速遮色片模式」？
「快速遮色片模式」讓你能夠像繪圖一樣編輯選取範圍，比傳統的選取工具更精細、可控制。
你可以使用筆刷、漸層、填色等工具**「畫出」選取範圍**，對細節處理（如頭髮邊緣、柔邊物件）特別有用。

✅ 如何使用快速遮色片模式
啟用快速遮色片：
點選影像視窗左下角的小矩形圖示（在捲軸模式旁邊）。
或使用快捷鍵 Shift + Q。

畫面變化：
出現一層紅色半透明的覆蓋層：
紅色部分 = 未被選取的區域
透明/原圖可見的部分 = 已被選取的區域

開始編輯：
使用 筆刷工具：
用 黑色 畫代表從選取中排除。
用 白色 畫代表加入選取範圍。
灰色則表示部分選取（可產生柔邊效果）。
也可使用漸層、填滿、橡皮擦等工具來處理選取範圍。
退出快速遮色片：
再次按下 Shift + Q。
編輯好的遮色片會自動轉換為標準的選取範圍（出現虛框）。

🧠 小技巧
適用於處理像是頭髮、羽毛、模糊邊緣、光暈效果。
配合 高斯模糊（Gaussian Blur） 可讓邊緣更自然柔和。
可將選取儲存為通道：選單 → 選取 → 儲存至通道，方便日後重複使用。

🔃 Google Sheets 表格排序與篩選（Sorting & Filtering）

🎯 教學目標
學會對資料表進行欄位排序
理解篩選（Filter）與條件篩選的差異
建立可互動的資料檢視工具

一、📊 排序（Sort）基礎操作

選取整個資料區塊（包含欄位標題）
點選 資料 → 建立範圍內排序（Sort range）
勾選「資料有標題列」
選擇你要排序的欄位與方向（升序/降序）

📌 範例：
姓名	分數
張三	85
李四	90
王五	75
→ 排序「分數」欄為降序後，李四會排最上面。

二、🔎 資料篩選（Filter）應用

✅ 開啟篩選
選取標題列（例如第 1 列）
點選 資料 → 建立篩選條件（Create a filter）
每欄會出現小漏斗圖示

✅ 篩選操作
點擊漏斗圖 → 可選取要顯示的值（✔ 勾選特定項目）
可選擇條件：
數值大於 / 小於 / 等於
文字包含 / 不包含
日期在範圍內

📌 範例任務：
篩選出「分數大於 80」的學生
只顯示「未繳費」的客戶
篩選出日期為本月的紀錄
三、📁 篩選 vs. 排序 差異

項目	排序	篩選
目的	調整資料順序	隱藏不符合條件的資料
結果	所有資料都在，只是順序變	暫時隱藏部分資料
是否會改動整表	會移動資料列順序	不改變原始順序
結合條件用法	一次排序一欄或多欄	可設定複雜條件（如大於、文字含特定字）

🔹 必學技巧與功能
功能	快捷鍵 / 操作	說明
開新視窗	Ctrl + N	開啟另一個總管視窗
返回 / 前進	Alt + ← / →	類似瀏覽器的「返回上一層」
進階搜尋	使用 *, ?, date:, type: 等條件	例：搜尋所有 PDF *.pdf 或日期為 2024 年的檔案 date:2024
選擇多個檔案	Shift 或 Ctrl 搭配滑鼠	批次操作檔案
預覽窗格	Alt + P	可預覽圖片、PDF、文件內容（需支援）
顯示/隱藏副檔名	功能區 →「檢視」→「副檔名」勾選	預設可能隱藏副檔名，要顯示更安全
顯示隱藏檔案	功能區 →「檢視」→「隱藏的項目」勾選	檢查 .ini、.sys 等系統檔時有用
批次重新命名	選多個檔案 → 按 F2	所有檔案會自動加上編號
快速存取清單	拖曳資料夾至左側導覽列的「快速存取」	常用資料夾一鍵到達

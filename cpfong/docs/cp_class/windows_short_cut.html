✅ 常用 Windows 鍵 + 鍵盤快捷鍵一覽表

快捷鍵	功能說明
Win + E	Explorer 開啟 檔案總管
Win + R Run 開啟「執行」
Win + M Minimize 最小化所有視窗

Win + Shift + S	Select 開啟螢幕截圖工具
Win + V	開啟剪貼簿歷史（需先啟用）
Win + I	Information 開啟「設定」
Win + +/-	放大鏡放大 / 縮小
Win + S 或 Win + Q	開啟搜尋

Win + D Desktop	顯示桌面（再次按下可還原）
Win + L	Lock 鎖定電腦
Win + A	開啟「通知中心 / 快速設定」
Win + X	開啟進階功能表（等同右鍵點開始按鈕）
Win + Tab	開啟「工作檢視」（多工作桌面）

Win + Ctrl + D	新增一個虛擬桌面
Win + Ctrl + 左/右 切換虛擬桌面
Win + Ctrl + F4	關閉目前虛擬桌面

Win + . 或 Win + ;	開啟表情符號選單（Emoji）
Win + P	投影螢幕選單（切換第二螢幕）
Win + Number	啟動固定在工作列上的第 N 個程式
Alt + Tab	切換視窗（快速視窗切換）
Ctrl + Shift + Esc	直接開啟工作管理員

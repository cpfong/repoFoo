✅ 步驟：新增英文（美式）輸入法
開啟設定
	按下 Win + I 鍵 → 開啟「設定」
	選擇【時間與語言】
	點選左側的「時間與語言」（Time & Language）
	點選【語言與區域】
	選擇「語言與區域」（Language & Region）

新增語言
	點選「新增語言」按鈕（Add a language）
	搜尋 English (United States)，然後點選 下一步
	確認安裝語言套件（可以不用勾選語音辨識或其他選項），然後按「安裝」

切換輸入法
	安裝完成後，任意輸入時你可按 Win + Space（或 Ctrl + Shift）快速切換輸入法。
	（選用）調整預設輸入法順序
	回到「語言與區域」頁面
	點選「Preferred languages」中的項目，使用上下箭頭調整英文輸入法的順序

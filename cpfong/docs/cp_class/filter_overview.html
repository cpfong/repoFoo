在 GIMP 中，Filter（濾鏡） 是圖像處理的核心工具之一，它的作用可以總結成：
對圖像應用各種視覺效果、修飾技術、紋理處理或自動調整演算法。

濾鏡讓你能夠：
✅ 1. 美化圖像
模糊（Blur）柔化肌膚、背景
銳化（Sharpen）讓邊緣更清晰
陰影與光暈（Drop Shadow, Glow）提升視覺層次

✅ 2. 修復或處理照片問題
移除雜訊（Noise Reduction）
修正透視、鏡頭畸變（Lens Distortion）
模擬景深、模糊背景（Gaussian Blur）

✅ 3. 創造特效
藝術風格：油畫、水彩、素描（Artistic Filters）
扭曲、鏡射、漣漪、玻璃效果（Distort）
動態模糊、運動模糊（Motion Blur）

✅ 4. 合成與影像處理
高反差保留（High Pass）→ 用於銳化遮罩
色彩調整輔助工具（如 Channel Mixer）


🧪 GIMP 濾鏡範例分類



類別常見用途

Blur 模糊	柔化背景、做柔焦人像、磨皮

Enhance 強化	利化、去雜訊、修邊緣

Distorts 扭曲做藝術效果，如波紋、漩渦、玻璃、鏡像

Light & Shadow	投影、發光、光暈、環境光效果

Render 繪製產生雲朵、火焰、雜訊圖、格紋等圖樣

Artistic 藝術風格油畫、水彩、鉛筆素描、漫畫風

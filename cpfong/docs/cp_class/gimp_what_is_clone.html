在 GIMP 中，Clone Tool（複製工具） 是一個用來**「複製一部分圖像內容到另一個區域」**的繪圖工具，功能就像「圖像中的複製貼上筆刷」。

🧰 Clone Tool 是什麼？
Clone Tool（快捷鍵 C） 可讓你從圖像中選擇一個「來源點」，然後把該處像素畫刷式地塗抹到其他地方。
你畫的每一筆，其實都是在「貼來源的內容」。

🎯 Clone Tool 用來做什麼？
✅ 主要用途：
用途	說明
修復照片缺陷	去除皮膚瑕疵、雜物、背景中的干擾物（如電線、雜點）
複製自然紋理	如草地、天空、頭髮、皮膚紋理等，自然補洞或延伸
填補去背後的破口	補回不自然的邊緣、斷裂的髮絲、透明區域
藝術創作或圖像拼貼	重複花紋、圖案、素材做視覺設計
🧪 例子：
🎨 例 1：人像照片修圖

用 ⌘+點 選擇乾淨肌膚 → 刷掉痘痘或黑斑。
🌿 例 2：風景照修補

複製藍天 → 覆蓋天空中的電線。
💇 例 3：人物去背精修髮絲

複製內部髮絲區 → 補刷邊緣斷髮、模糊背景邊界。

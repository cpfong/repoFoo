| 操作功能           | Windows / Linux       | macOS                            |
| -------------- | --------------------- | -------------------------------- |
| 找資料 | Ctrl + F | Command + F 
| 移到第一格 A1       | `Ctrl + Home`         | `Fn + Command (⌘) + ←`           |
| 移到最後一格（含資料的右下） | `Ctrl + End`          | `Fn + Command (⌘) + →`           |
| 向上 / 下捲動一頁     | `Page Up / Page Down` | `Fn + ↑ / Fn + ↓`                |
| 移到資料區最上方（同欄）   | `Ctrl + ↑`            | `Command (⌘) + ↑`                |
| 移到資料區最下方（同欄）   | `Ctrl + ↓`            | `Command (⌘) + ↓`                |
| 移到資料區最左方（同行）   | `Ctrl + ←`            | `Command (⌘) + ←`                |
| 移到資料區最右方（同行）   | `Ctrl + →`            | `Command (⌘) + →`                |
| 移動到下一張工作表      | `Ctrl + Page Down`    | `Option (⌥) + Command (⌘) + →`   |
| 移動到上一張工作表      | `Ctrl + Page Up`      | `Option (⌥) + Command (⌘) + ←`   |
| 選取連續儲存格（橫向/直向） | `Shift + 方向鍵`         | 同上                               |
| 跳至任意儲存格（如 G20） | `Ctrl + G` 或 `F5`     | `⌘ + Option + Shift + H`（非預設需自設） |

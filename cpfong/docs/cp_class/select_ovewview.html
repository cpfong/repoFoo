| 工具名稱                              | 快捷鍵      | 用途概述
| ------------------------------------- | ----------- | -------------------------
| **1. Rectangle Select Tool**          | `R`         | 選取矩形區域
| **2. Ellipse Select Tool**            | `E`         | 選取橢圓或圓形區域
| **3. Free Select Tool** (Lasso)       | `F`         | 手繪或點選方式自由圈選區域
| **4. Fuzzy Select Tool** (Magic Wand) | `U`         | 根據顏色相近區域進行連續選取
| **5. Select by Color Tool**           | `Shift + O` | 選取所有與指定點**顏色一致的區塊**（不必相連
| **6. Scissors Select Tool**           | `I`         | 智慧剪刀：依邊緣輪廓自動產生選區（適合人像或物體
| **7. Foreground Select Tool**         | `G`         | 半自動分離前景與背景，用於抠圖
| **8. Select by Path Tool**            | N/A         | 使用路徑（Paths）轉成選區，適合精細控制
| **9. Quick Mask Mode**                | `Shift + Q` | 以遮色片形式編輯選區（畫筆/橡皮擦微調選取範圍

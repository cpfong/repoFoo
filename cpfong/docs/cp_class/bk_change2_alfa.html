新增 Alpha 通道（讓透明度可用）
        右鍵點擊圖層面板中的「Background」圖層。
        選擇「Add Alpha Channel（新增 Alpha 通道）」，如果這個選項是灰色的，就代表已經加過，可以略過。

    選取白色區域
        使用選取工具（快捷鍵 Shift + O 是魔術棒選取工具）。
        在白色背景上點一下，它會選取整個白色區域。
        如果白色範圍沒完全選到，可調整「閾值（Threshold）」數值以擴大選取範圍（在工具選項中）。

    刪除選取區域
        按 Delete 鍵（或選單：Edit → Clear）。
        白色會被刪除，變成透明的方格背景。

    取消選取
        Select → None（或快捷鍵 Shift + Ctrl + A）。

    儲存為支援透明度的格式
        檔案 → 匯出為（Export As），選擇 PNG 格式（JPG 不支援透明）。

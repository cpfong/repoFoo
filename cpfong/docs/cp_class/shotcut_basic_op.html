✂️ 1. 分割影片（Split at Playhead）
用途：將影片切成兩段以便移動、刪除、套用不同濾鏡。

操作步驟：
將影片拖入時間軸（Timeline）
移動播放頭（Playhead）到你要切的位置
按下工具列的 ✂️ Split 按鈕（或快捷鍵 S）


⛏️ 2. 刪除片段
用途：移除不需要的部分。
操作步驟：

點選片段 → 按 Delete 鍵
空白區段會留下，你可以之後合併或套其他片段


🧱 3. 移動／重排片段
操作步驟：

點選片段 → 拖曳到新位置
若拖曳片段讓它與前後片段對齊，會自動吸附（snapping）


🧲 4. 啟用/停用吸附（Snapping）
用途：讓片段靠近時自動對齊，剪輯更精確。
操作步驟：

工具列中找到磁鐵圖示 🧲（Snapping）
點一下啟用/取消

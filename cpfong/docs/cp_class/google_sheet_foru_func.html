公式（Formula）

定義：以 = 開頭的計算式，例如：=A1 + B1
可包含欄位、數字、運算符號
🔧 函數（Function）

定義：Google Sheets 提供的內建計算工具，例如：=SUM(A1:A5)
結構：=函數名稱(參數)

📌 常見運算符號
符號	意義
+	加
-	減
*	乘
/	除
&	串接文字
=	等於（在邏輯運算中）

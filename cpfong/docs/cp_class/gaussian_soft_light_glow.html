🎯 製作「柔光遮罩（Soft Light Glow）」效果
這種效果常用於人像、商品或藝術照，用來營造光暈、柔光、氛圍感、電影感。


🖼️ 實際情境：
你有一張人物或靜物照片，希望加上一圈柔和光暈、讓主體更加突出，整體氣氛更夢幻柔美。

✅ 操作流程：用 Gaussian Blur 製作柔光遮罩
1️⃣ 複製圖層
右鍵 → Duplicate Layer（複製主圖層）
確保模糊處理不影響原始細節

2️⃣ 對副本套用 Gaussian Blur
選擇上層副本圖層
選單：Filters → Blur → Gaussian Blur
Radius 建議值：
肖像圖：15–40 px
景物照：40–100 px
整張圖會變糊，但這是刻意的

3️⃣ 改變圖層混合模式為「Soft Light」或「Overlay」
在圖層面板中，將模糊圖層的 Mode 設為：
Soft light：柔和自然、氣氛感強
Overlay：更亮、更銳利
💡 效果會讓主體「發光」，但保留原始圖細節。

4️⃣（選用）加遮色片控制光暈區域
加上遮片（白色）
用黑色柔邊筆刷刷掉不想要光暈的地方（例如人物五官）
可以讓光集中在四周或背景，主體保清晰

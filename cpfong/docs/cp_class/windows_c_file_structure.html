📁 **C:**（磁碟根目錄）
整個系統的頂層目錄。包含作業系統、安裝的程式、使用者資料、暫存檔等。

📂 C:\Windows
🔧 Windows 作業系統核心檔案
包含系統元件、驅動程式、視窗介面（如檔案總管）、字型、更新檔等。
⚠️ 不建議手動修改或刪除，可能導致系統故障。


📂 C:\Program Files
📦 預設的 64 位元應用程式安裝資料夾
安裝一般軟體時的預設位置。
系統會限制權限，需管理員身分操作。


📂 C:\Program Files (x86)
📦 32 位元應用程式的安裝目錄（僅 64 位元系統會有）
為了相容性，將 32 位元軟體與 64 位元分開儲存。


📂 C:\Users（使用者）
👤 各個使用者的個人資料夾
每位使用者會有一個子目錄，如 C:\Users\cp。
包含桌面、文件、下載、圖片、音樂等資料夾。
應用程式的使用者設定檔也會儲存在此處（如 Chrome 書籤、VSCode 設定等）。

檢視與載入佈局（View / Load Layout）
1. 開啟 View 選單
路徑：View → Load Layout
功能：讓使用者可以快速切換、載入不同的工作區配置。

2. 預設佈局選項（內建 Layout）
Kdenlive 預設內建幾種工作區佈局，方便不同類型的剪輯工作：
Editing：標準剪輯用介面。
Audio：音訊編輯專用（顯示 Audio Mixer 等）。
Color：顏色校正（顯示 Scopes、色彩面板）。
Logging：素材整理階段使用。

3. 儲存自訂佈局
路徑：View → Save Layout As...
可輸入自訂名稱，例如「講義剪輯用」、「單螢幕版面」等。
自訂後會出現在 Load Layout 清單中，日後可快速切換。

4. 還原預設佈局
路徑：View → Reset Layout
若視窗亂掉或誤關面板，可一鍵回到預設狀態。

5. 拖拉重組
面板可自由拖拉、合併、分離、停駐。
每次變更都會即時記錄在目前的 Layout 狀態中。

📱 在 iPhone 上匯入 KML/KMZ 檔案至 MAPS.ME

MAPS.ME 支援匯入 KML/KMZ 格式的書籤與路線。步驟如下：
    準備 KML/KMZ 檔案
        可以透過電子郵件、iCloud Drive、AirDrop 等方式將檔案傳到 iPhone。

    在 iPhone 上開啟檔案
        透過「檔案」App 或「郵件」App 找到該檔案。
        點一下並選擇「分享」→ 選擇 MAPS.ME 開啟（若未出現，請確認已安裝並更新 MAPS.ME）。

    完成匯入
        MAPS.ME 會自動啟動並匯入檔案。
        匯入成功後，請點選畫面底部的「星號」圖示即可看到新增的書籤。

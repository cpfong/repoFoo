當你匯入純文字檔（如 .txt 或 .csv） 到 Google Sheets 時，系統會自動推測（guess/infer）每一欄的資料格式，並根據內容進行格式轉換，這是 Google Sheets 的預設行為。

🧠 自動推測與格式轉換行為
檔案內容示例	系統推測格式	結果
123	數字	顯示為 123，右對齊
2025/06/11	日期	顯示為日期格式
0123	數字	顯示為 123（開頭 0 消失）
0912345678	數字	可能變成 9.12E+08（科學記號）
TRUE / FALSE	布林值	顯示為邏輯值
=A1+B1	公式	被視為字串而非公式

⚠️ 問題：這些「推測」常造成誤判或資料失真
例如：

郵遞區號 02105 → 變成 2105
手機 0912345678 → 被科學記號表示
原本是 ID 或學號 → 被當作數字處理

✅ 解法：在匯入時關閉自動格式推測
當你用 「檔案 → 匯入（File → Import）」 匯入 .csv 或 .txt 檔時：
選擇檔案 → 點選「匯入位置（Import location）」與「分隔符號」
看是否出現「轉換文字為數字與日期（Convert text to numbers, dates, and formulas）」

✅ 預設為「打勾」
❌ 請取消勾選這個選項，讓所有資料都以文字方式匯入，不做自動格式推論

若你從 Google Drive 直接開啟 .csv 檔，系統不會給你這個選項，會直接推斷格式。

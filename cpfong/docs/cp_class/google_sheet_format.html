當我們談到 Google Sheets 的格式設定（formatting），重點是控制 儲存格的顯示方式，不改變原始資料值。這對於提升可讀性、強調重點與視覺呈現都非常重要。

一、進入格式功能的位置
點選上方選單「格式（Format）」或使用工具列上的快速圖示（如 B：粗體，斜體，顏色、靠齊等）

二、常見格式類型
類型	用途與說明	範例
數字（Number）	預設數值格式。可顯示小數點、負號等	1234.56
貨幣（Currency）	加上貨幣符號並自動小數點對齊	$1,234.56 / NT$1,234.56
會計（Accounting）	類似貨幣，但貨幣符號對齊欄位左側	更整齊的帳務報表
百分比（Percent）	自動乘 100 並加上 % 號	輸入 0.85 → 顯示為 85%
日期 / 時間	支援多種格式，例如 YYYY/MM/DD	2025/06/11
文字（Plain Text）	將內容當作文字處理，不解析成數字或函數	儲存郵遞區號、零開頭學號

三、視覺格式調整（工具列上操作）
類型	說明與用途
字型粗細、大小、顏色	強調某些資料區塊
儲存格底色	區分不同區域，例如標題列
框線與邊框	建立表格感，提升整齊度
文字對齊	左 / 中 / 右對齊、垂直置中

四、條件格式（Conditional formatting）
非常實用！依據儲存格內容自動改變顏色或樣式。

操作步驟：
選取範圍（例如 A2:A100）
點選「格式 → 條件格式」
設定條件（如：小於 60 → 顯示紅色字）
點選「完成」

常見案例：
分數低於及格線顯示紅色
出席狀況為 "缺席" 的列整行變灰
達標金額以綠色突出顯示


五、格式複製貼上（Paint Format）
選好一個格式好的儲存格 → 點選工具列上的「油漆刷」→ 點選要套用的儲存格
適合快速統一格式風格

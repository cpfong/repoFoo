使用「路徑工具」移除背景步驟

✅ 準備
    確認圖像有 Alpha 通道：
    圖層 → 透明 → 加入 Alpha 通道（Add Alpha Channel）

① 啟用「路徑工具」
    工具箱選「🖊️ 路徑工具」或按快捷鍵 B（Bézier）。
    在圖像上沿著主體輪廓逐點點選建立路徑（可用 Ctrl+點擊調整曲線）。

② 封閉路徑
    點回第一個節點使其「連線成封閉區域」。
    按 Enter 鍵將路徑轉為選取區域（你會看到螢光線框）。

③ 刪除背景
    選取 → 反轉（Select → Invert）
    按 Delete 鍵刪除背景部分。

④ 微調（可選）
    使用 選取 → 羽化（Feather） 讓邊緣更自然。
    或建立「遮色片」，可反覆編修遮罩細節。

高斯模糊（Gaussian Blur） 是 GIMP 中最常見的模糊濾鏡之一，用於產生柔和效果、背景虛化、陰影柔化、甚至仿景深效果。

📘 原理簡介

「高斯模糊」是基於統計學中的高斯分布（鐘形曲線）計算像素鄰近值的加權平均，模糊邊緣但保留圖像的整體結構。
🧰 操作方法

    選擇要模糊的圖層、區域或物件（例如圖層、選取範圍或圖層蒙版）。
    選單：濾鏡 (Filters) → 模糊 (Blur) → 高斯模糊 (Gaussian Blur)
    在彈出的對話框中：
        水平模糊半徑 (Horizontal Radius)：橫向模糊程度
        垂直模糊半徑 (Vertical Radius)：縱向模糊程度
        兩者可設相同值，也可不等（創造水平流動感）
    預覽無誤後按 確定 (OK)。

💡 常見用途與技巧
用途	說明
背景虛化	對背景圖層應用高斯模糊，模擬相機淺景深效果
柔化邊緣	在選取範圍邊緣羽化後再模糊，可使切割更自然
製作光暈/發光	將白色圖案複製一層後模糊，再疊加於下方圖層
陰影效果	建立陰影層，填色後套用高斯模糊，最後往下方偏移
仿玻璃模糊	小範圍高斯模糊可模擬磨砂玻璃效果，搭配不透明度更佳

🛠 範例推薦
    從 Unsplash 下載一張街景或肖像照片。
    將人物選出後建立一個背景圖層並模糊。
    調整上層圖像的邊緣，做出類似「人像攝影」的效果。

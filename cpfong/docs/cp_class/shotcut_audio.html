在 Shotcut 中加入與管理「音效」是剪輯中的基本技巧，無論是背景音樂、環境音、還是效果聲，都屬於音效處理範疇。


🎧 常見音效處理需求：
功能	說明
插入音效	將 MP3、WAV 等檔案拖曳進時間軸（Timeline）
調整音量	使用 音量/增益（Volume/Gain）濾鏡
淡入淡出音效	使用 淡入（Fade In） / 淡出（Fade Out） 濾鏡
靜音某段音效	使用音量濾鏡設為 -∞ dB，或用剪刀剪掉該段
同步音效與畫面節奏	手動移動音效片段，搭配 波形圖 對齊關鍵節點
多段音效混音	使用多條 音訊軌道（Audio Tracks） 疊加管理


🔧 實作步驟範例（插入＋調整音效）：
加入音訊軌道：
點選左下角 + ➜ Add Audio Track
拖曳音效檔至音訊軌道
支援 .mp3, .wav, .ogg 等常見格式
選取音效片段 ➜ 開啟 Filters
加入 Volume / Gain 來調整音量
加入 Fade In Audio, Fade Out Audio 做進出效果
視覺對齊：
使用 View > Show Audio Waveform 方便對拍與節奏處理


🧠 小技巧：
可在片段之間加入音效來模擬「轉場聲效」。
音樂與旁白應放在不同音訊軌上，方便獨立調整。
若需要精確控制，建議使用「關鍵影格（Keyframes）」配合音量濾鏡動畫。

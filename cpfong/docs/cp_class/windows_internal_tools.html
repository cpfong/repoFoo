🧩 必懂的 Windows 內建工具
檔案總管	explorer.exe	開啟檔案瀏覽器，也可用於啟動資料夾或捷徑
記事本	notepad.exe	簡易文字編輯器，在新版 Windows 可選安裝或由商店版本取代
小畫家（Paint）	mspaint.exe（原為 pbrush.exe）	基本繪圖工具
命令提示字元	cmd.exe	傳統的命令列介面
PowerShell	powershell.exe 或 pwsh.exe we.exe (windows terminal) 現代化指令與腳本環境
工作管理員	taskmgr.exe	查看系統執行中程序與效能
執行視窗	Win + R	非獨立程式，但用來快速啟動其他工具
遠端桌面連線	mstsc.exe	遠端桌面用戶端（限 Windows Pro/Enterprise）
剪取工具	snippingtool.exe 或 Win + Shift + S	擷圖工具，新版為「剪取與繪圖」
控制台	control.exe	傳統設定介面（部分功能仍依賴） replaced with W+i (info)

登錄編輯器	regedit.exe	編輯 Windows 註冊表（請小心使用）
系統組態	msconfig.exe	啟動項目與系統啟動選項（Windows 10 起較少用）
服務管理員	services.msc	管理背景服務
事件檢視器	eventvwr.msc	查看系統與應用程式記錄
裝置管理員	devmgmt.msc	管理硬體與驅動程式
磁碟管理	diskmgmt.msc	磁碟分割與磁碟狀態
資源監視器	resmon.exe	詳細系統資源監控
系統資訊	msinfo32.exe	顯示系統摘要與硬體資訊
放大鏡	magnify.exe	螢幕放大輔助工具

📂 Desktop (桌面)
🖥️ 你桌面上看到的檔案或捷徑都儲存在這裡。
任意拖曳到桌面的檔案，實際就是存在這個資料夾。


📂 Documents（文件）
📄 儲存 Word、Excel、PDF、記事本等文件的預設位置。
很多應用程式會自動把存檔預設到這裡。


📂 Downloads（下載）
🌐 從瀏覽器或其他程式下載的檔案預設都會放在這裡。


📂 Pictures（圖片）
🖼️ 儲存圖片、相片、截圖（部分情況下）等。


📂 Videos（影片）
📹 儲存影片檔，例如手機傳輸過來的 MP4，或螢幕錄影。


📂 Music（音樂）
🎵 儲存 MP3、WAV 等音樂檔案。

📂 AppData（⚠️ 預設為隱藏）
🧠 應用程式資料與設定儲存位置。分三部分：(很多軟體的自訂設定、暫存檔、快取資料、登入憑證都藏在這裡)



子資料夾	功能說明

Local 	僅限本機使用的資料與快取（e.g. 瀏覽器資料)

LocalLow	安全限制較高的應用程式儲存資料（如 Flash)

Roaming 	可隨使用者帳戶漫遊的資料（e.g. Outlook、VSCode 設定）

功能：用畫筆標出前景，GIMP 幫你抠圖
步驟：
圈出大致區域
塗前景區域
GIMP 分析並建立精準選區
適合：背景雜亂但主體明顯的圖像
note: ENTERs


🧪 使用步驟：
    選取工具
        工具箱中選取：🖌️「前景選取工具」
        或：工具 → 選取工具 → 前景選取

    框選前景所在範圍 (要下一個 ENTER 才會變藍 enter-1/3)
        用滑鼠畫一個大致包住主體的區域（矩形或粗略曲線）
        放開滑鼠後，背景會變藍（代表預設是背景）

    塗抹前景細節
        接下來滑鼠會變成畫筆
        在主體上塗幾筆（建議多塗幾個不同顏色的區塊）
        GIMP 會根據你的塗抹區域，計算出整個主體區域

    按下 Enter (筆的 enter 共有二次, 第一次(enter-2/3)下完立即運算. 後再出現筆這時要再下 enter (enter-3/3)
        系統分析後，會轉成一個選取區（螢光線圍起來）
       ✅ 現在你已選到主體，可以複製、剪下、刪除背景等操作

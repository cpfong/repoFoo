🧭 二、三種參照方式與語法
類型	語法範例	意義	拖曳時會怎樣？
相對參照	A1	A 欄第 1 列，欄與列都會隨拖曳變動	拖到下一列 → A2
絕對參照	$A$1	永遠指向 A1，欄與列都鎖定	拖到哪都還是 $A$1
混合參照	$A1 或 A$1	鎖欄不鎖列 / 鎖列不鎖欄	依拖曳方向部分固定
🧪 三、範例說明
假設你在 C2 輸入：

=A2*B2
接著將此格公式向下拖曳幾列 → 系統自動轉成：

C3: =A3*B3
C4: =A4*B4
→ 這就是「相對參照」（欄與列都跟著變）

如果你改寫成：

=A2*$B$1
→ 拖曳後，只有 A2 部分變，$B$1 永遠指向 B 欄第 1 列

🧠 快捷鍵提示
在編輯儲存格公式時，選取某一儲存格參照，按下：
F4（Windows）
Command + T（Mac）
會在 A1、$A$1、A$1、$A1 間循環切換
